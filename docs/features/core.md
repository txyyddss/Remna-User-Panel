# Основные возможности

Minishop закрывает путь от регистрации пользователя до оплаты, продления, поддержки и сопровождения подписки.

## Для пользователей

- Регистрация через [Telegram Mini App](telegram-auth.md) или [email-код](email-login.md).
- Просмотр подписки, срока действия, трафика и ссылки подключения.
- Покупка подписки, пакетов трафика и дополнительных устройств.
- Пробный период, промокоды и реферальные сценарии.
- Тикеты поддержки внутри Mini App.
- Встроенные инструкции установки и публичные ссылки `/s/<token>`.

## Для администраторов

- Поиск и управление пользователями.
- Настройка платежей, тарифов, внешнего вида и поддержки.
- Рассылки, промокоды и логи действий.
- Ручная синхронизация с Remnawave Panel.
- Редактор JSON-каталога тарифов.

Подробности: [админ-панель](admin-panel.md), [Mini App](web-app.md) и [поддержка пользователей / тикеты](support.md).
